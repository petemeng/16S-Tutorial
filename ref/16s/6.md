## 从 QIIME2 到 R：导出数据

QIIME2 的 `.qza` 文件不能被 R 直接读取。我们需要导出为标准格式，然后用 R 包 `qiime2R` 或手动导入的方式构建 phyloseq 对象。

最省事的方法是用 `qiime2R` 包——它可以直接读 `.qza` 文件。

```r
# ============================================================
# 文件：analysis/06_composition_visualization.R
# 功能：物种组成可视化（堆叠柱状图、热图）
# 输入：QIIME2 的 .qza 文件
# 输出：发表级物种组成图
# ============================================================

# 安装 qiime2R（如果尚未安装）
# install.packages("devtools")
# devtools::install_github("jbisanz/qiime2R")

library(qiime2R)
library(phyloseq)
library(microeco)
library(ggplot2)
library(dplyr)
library(tidyr)
library(readr)
library(RColorBrewer)
```

```r
# ============================================================
# Step 1: 用 qiime2R 直接读取 QIIME2 artifact
# 一步构建 phyloseq 对象——后续所有 R 分析都基于它
# ============================================================

ps <- qza_to_phyloseq(
    features = "results/table-filtered.qza",
    tree     = "results/rooted-tree.qza",
    taxonomy = "results/taxonomy.qza",
    metadata = "data/sample-metadata.tsv"
)

cat("phyloseq 对象摘要：\n")
ps
```

670 个 ASV、34 个样本、7 个分类学等级（域到种）、系统发育树——全部装在一个 phyloseq 对象里了。后续所有操作都围绕这个对象展开。

💡 **如果 qiime2R 装不上**

> `qiime2R` 依赖一些编译型包，在某些服务器上可能安装失败。备选方案是手动导入：
>
> ```bash
> # 在 QIIME2 环境中先导出为标准格式
> qiime tools export --input-path results/table-filtered.qza --output-path export/
> qiime tools export --input-path results/taxonomy.qza --output-path export/
> qiime tools export --input-path results/rooted-tree.qza --output-path export/
> biom convert -i export/feature-table.biom -o export/feature-table.tsv --to-tsv
> ```
>
> 然后在 R 中用 `phyloseq::import_biom()` 或直接读 TSV 手动构建 phyloseq 对象。

## 门水平堆叠柱状图

这是微生物组论文中出现频率最高的图。每个样本一根柱子，不同颜色代表不同的门（Phylum），高度代表相对丰度。

```r
# ============================================================
# Step 2: 门水平堆叠柱状图
# 思路：
#   1. 把 ASV 水平的 counts 合并到门水平
#   2. 转换为相对丰度
#   3. 只保留丰度 top 的门，其余合并为 "Others"
#   4. 按体位点分面展示
# ============================================================

# 合并到门水平
ps_phylum <- tax_glom(ps, taxrank = "Phylum")

# 转换为相对丰度
ps_phylum_rel <- transform_sample_counts(ps_phylum,
                                          function(x) x / sum(x))

# 转换为长格式数据框
phylum_df <- psmelt(ps_phylum_rel) %>%
    mutate(Phylum = as.character(Phylum))

# 找到 top 8 的门（按所有样本的平均丰度排序）
top_phyla <- phylum_df %>%
    group_by(Phylum) %>%
    summarize(mean_abund = mean(Abundance)) %>%
    arrange(desc(mean_abund)) %>%
    head(8) %>%
    pull(Phylum)

# 其余的门合并为 "Others"
phylum_df <- phylum_df %>%
    mutate(Phylum_plot = ifelse(Phylum %in% top_phyla,
                                 Phylum, "Others"))

# 定义配色——用专门为微生物组设计的色板
phylum_colors <- c(
    "Bacteroidota"     = "#E64B35",
    "Firmicutes"       = "#4DBBD5",
    "Proteobacteria"   = "#00A087",
    "Actinobacteriota" = "#3C5488",
    "Fusobacteriota"   = "#F39B7F",
    "Verrucomicrobiota"= "#8491B4",
    "Cyanobacteria"    = "#91D1C2",
    "Patescibacteria"  = "#DC9C26",
    "Others"           = "grey75"
)

cat("Top 8 门（按平均相对丰度）：\n")
phylum_df %>%
    filter(Phylum_plot != "Others") %>%
    group_by(Phylum_plot) %>%
    summarize(mean_pct = round(mean(Abundance) * 100, 1)) %>%
    arrange(desc(mean_pct)) %>%
    print()
```

<!-- 图 1 位置：门水平堆叠柱状图 -->

**图 1：门水平物种组成堆叠柱状图。** 样本按体位点分面展示。肠道（gut）以 Bacteroidota（红色）和 Firmicutes（蓝色）为主；手掌以 Proteobacteria（绿色）和 Firmicutes 为主；舌头呈现 Proteobacteria、Bacteroidota 和 Firmicutes 的三分格局。

**从这张图能直接读出的生物学信息：**

**肠道**是"拟杆菌-厚壁菌双主导"模式——这是人类肠道微生物组最经典的特征，在全球不同人群中高度一致。Bacteroidota:Firmicutes 的比例（B/F ratio）曾被认为和肥胖有关，虽然这个观点现在有争议，但它的主导地位是无可争议的。

**手掌**以 Proteobacteria 为主——变形菌门中的很多物种是环境中常见的耐干燥细菌，适合皮肤表面这种暴露、干燥、不断受到扰动的环境。

**同一体位点内部的组成很稳定**——同一个面板内的柱子颜色比例大致相似。这和第 5 篇 PERMANOVA 的结论一致：组内变异远小于组间变异。

⚠️ **踩坑预警：堆叠柱状图展示的是相对丰度，不是绝对丰度**

> 这张图的每根柱子都加到 100%。这意味着你看到的"Bacteroidota 在肠道中占 35%"是相对于该样本中所有细菌的比例——不是绝对菌量。
>
> 一个常见的误读：如果某种菌在处理组中的相对丰度从 10% 升到 20%，你可能会说"这种菌增多了"。但也可能是**其他菌减少了**，导致这种菌的**比例**被动上升，而它的绝对数量其实没变甚至在减少。
>
> **在论文讨论中，用"相对丰度增加"而不是"丰度增加"来描述——除非你有绝对定量的数据（如 qPCR 或掺入标准品）。**

## 属水平堆叠柱状图

门水平看的是大画面，属水平看的是细节。

```r
# ============================================================
# Step 3: 属水平堆叠柱状图
# 属水平的物种太多，只展示 top 15
# ============================================================

ps_genus <- tax_glom(ps, taxrank = "Genus", NArm = FALSE)
ps_genus_rel <- transform_sample_counts(ps_genus,
                                         function(x) x / sum(x))

genus_df <- psmelt(ps_genus_rel)

# 处理未注释到属的 ASV
genus_df <- genus_df %>%
    mutate(Genus = ifelse(is.na(Genus) | Genus == "",
                           paste0("Unclassified_", Family),
                           Genus))

# Top 15 属
top_genera <- genus_df %>%
    group_by(Genus) %>%
    summarize(mean_abund = mean(Abundance)) %>%
    arrange(desc(mean_abund)) %>%
    head(15) %>%
    pull(Genus)

genus_df <- genus_df %>%
    mutate(Genus_plot = ifelse(Genus %in% top_genera,
                                Genus, "Others"))

# 用更多颜色的色板
genus_colors <- c(
    colorRampPalette(brewer.pal(12, "Paired"))(15),
    "grey75"  # Others
)
names(genus_colors) <- c(top_genera, "Others")

p_genus <- ggplot(genus_df,
                   aes(x = Sample, y = Abundance,
                       fill = Genus_plot)) +
    geom_bar(stat = "identity", width = 0.9) +
    facet_grid(~ body.site, scales = "free_x", space = "free_x") +
    scale_fill_manual(values = genus_colors) +
    scale_y_continuous(labels = scales::percent,
                        expand = c(0, 0)) +
    labs(x = "", y = "Relative Abundance", fill = "Genus") +
    theme_songlab() +
    theme(
        axis.text.x = element_text(angle = 45, hjust = 1, size = 8),
        legend.position = "right",
        legend.text = element_text(face = "italic", size = 9),
        panel.spacing = unit(0.3, "lines")
    )

ggsave("results/figures/pub_genus_barplot.png", p_genus,
       width = 16, height = 7, dpi = 300)
```

<!-- 图 2 位置：属水平堆叠柱状图 -->

**图 2：属水平物种组成堆叠柱状图（Top 15 属）。** 属名用斜体（生物学规范）。肠道中 *Bacteroides*（拟杆菌属）占绝对优势，其次是 *Prevotella* 和 *Faecalibacterium*。手掌以 *Staphylococcus* 和 *Corynebacterium* 为主。舌头以 *Neisseria* 和 *Haemophilus* 为主。

💡 **属名为什么用斜体？**

> 生物学命名规范要求：属名和种名在正文和图中必须用斜体（*Bacteroides*, *E. coli*）。门名和更高层级用正体（Firmicutes, Proteobacteria）。审稿人会检查这个。在 ggplot2 中用 `element_text(face = "italic")` 即可。

## Top 物种热图

堆叠柱状图适合展示组成比例，热图适合展示"哪些物种在哪些样本中丰度高或低"的模式。

```r
# ============================================================
# Step 4: Top 30 属的热图
# 行 = 属，列 = 样本，颜色 = log 变换后的相对丰度
# ============================================================

# 提取 top 30 属
top30 <- genus_df %>%
    group_by(Genus) %>%
    summarize(mean_abund = mean(Abundance)) %>%
    arrange(desc(mean_abund)) %>%
    head(30) %>%
    pull(Genus)

# 构建矩阵
heatmap_df <- genus_df %>%
    filter(Genus %in% top30) %>%
    select(Sample, Genus, Abundance, body.site) %>%
    pivot_wider(names_from = Sample, values_from = Abundance,
                values_fill = 0)

# 转为矩阵
mat <- as.matrix(heatmap_df[, -c(1)])
rownames(mat) <- heatmap_df$Genus

# log10 变换（加伪计数避免 log(0)）
mat_log <- log10(mat + 1e-6)

# 样本注释
sample_meta <- genus_df %>%
    select(Sample, body.site) %>%
    distinct()
ann_col <- data.frame(
    Body_Site = sample_meta$body.site,
    row.names = sample_meta$Sample
)
ann_colors <- list(
    Body_Site = c("gut" = "#E64B35", "left palm" = "#4DBBD5",
                   "right palm" = "#00A087", "tongue" = "#3C5488")
)

pheatmap::pheatmap(
    mat_log,
    annotation_col = ann_col,
    annotation_colors = ann_colors,
    color = colorRampPalette(c("#F7FBFF", "#4292C6", "#08306B"))(100),
    cluster_rows = TRUE,
    cluster_cols = TRUE,
    show_colnames = FALSE,
    fontsize_row = 8,
    labels_row = paste0("italic('", rownames(mat_log), "')"),
    filename = "results/figures/pub_genus_heatmap.png",
    width = 12, height = 10
)

cat("属水平热图已保存\n")
```

<!-- 图 3 位置：属水平热图 -->

**图 3：Top 30 属的丰度热图。** 列按体位点聚类（顶部注释条），行为属。颜色深浅代表 log10 变换后的相对丰度。聚类结果清晰地将肠道与其他体位点分开，同时揭示了每个体位点的"特征菌属"模式。

**怎么读这张热图：**

列聚类自动把相似的样本归在了一起——注意顶部的颜色条，同一体位点的样本被完美聚合。这本身就是一个验证：说明微生物组成确实主要由体位点决定。

行聚类揭示了几个"物种模块"：一组属在肠道中高丰度但在其他部位低（*Bacteroides*、*Prevotella*、*Faecalibacterium*），另一组在手掌高丰度（*Staphylococcus*、*Corynebacterium*），第三组在舌头高丰度（*Neisseria*、*Haemophilus*、*Streptococcus*）。

## 本篇小结

这一篇我们完成了从 QIIME2 到 R 的完整过渡，并做了三种物种组成可视化：

**堆叠柱状图**（门/属水平）是微生物组论文的"标配图"。关键技巧：只展示 top N 的物种，其余合并为 Others；按分组分面展示；属名用斜体。

**热图**揭示了柱状图不容易看到的"物种-样本"对应模式——哪些属只在特定体位点出现，哪些属在所有体位点都有但丰度不同。

**分组平均图**把个体差异平滑掉，更适合做组间比较。

核心收获：

**phyloseq 是 R 微生物组分析的通用容器。** ASV 表 + 分类学 + 样本信息 + 系统发育树全部封装在一个对象里，所有分析函数都能从中取数据。构建好这个对象后，后续的多样性、差异分析、网络分析都不需要重新读数据。

**相对丰度 ≠ 绝对丰度——在论文措辞中务必区分。** 堆叠柱状图展示的是比例，不是绝对菌量。
