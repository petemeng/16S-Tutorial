## 本篇目标

前面十篇我们一直在研究"这里有什么"和"不同地方有什么不同"。现在换一个视角：**这些微生物是从哪里来的？**

Moving Pictures 数据集有四个体位点：肠道（gut）、舌头（tongue）、左手（left palm）、右手（right palm）。一个自然的问题是：手掌上的微生物组成，有多少可以用肠道或口腔来"解释"？换句话说——手掌上的细菌，有多大比例是从肠道"迁移"过来的（比如上完厕所后），有多大比例来自口腔（比如打哈欠时接触嘴巴），又有多少是手掌自己的"原住民"？

SourceTracker 就是回答这类问题的工具——它用贝叶斯混合模型估计每个"汇"（sink）样本的微生物组成中，来自各个已知"源"（source）的比例。

读完这一篇，你会：

1. 理解 SourceTracker 贝叶斯溯源模型的原理
2. 正确设置 source 和 sink 样本
3. 运行 SourceTracker2 并解读比例估计
4. 可视化溯源结果
5. 理解溯源分析的假设和局限

## 微生物溯源的原理

### 贝叶斯混合模型

SourceTracker 的核心假设很直觉：**每个 sink 样本是多个 source 的混合物。** 如果我们把手掌微生物组看作 sink，把肠道、口腔作为 source，那么手掌上观测到的物种组成可以被分解为：

```
手掌微生物 = α × 肠道贡献 + β × 口腔贡献 + γ × 未知来源
```

其中 α + β + γ = 1。SourceTracker 用吉布斯采样（Gibbs sampling）来估计这些混合比例。

**"未知来源"（unknown）很重要**——它代表了不能被任何已知 source 解释的部分。如果某个 sink 样本有 60% 的"unknown"，说明我们定义的 source 不够全面，还有大量微生物来自我们没纳入的源。

### 关键假设

1. **Source 样本代表了"纯"的源环境。** 如果你的肠道样本已经被其他环境污染了，估计结果就会失真。
2. **Sink 确实是 source 的混合物。** 如果 sink 有大量独特物种（不存在于任何 source 中），这些物种会被归入"unknown"。
3. **Source 之间应该有足够的区分度。** 如果两个 source 的物种组成太相似，模型会难以区分它们的贡献。

## Step 1：准备输入数据

SourceTracker 需要两个输入：**ASV 丰度表**（BIOM 格式）和一个**元数据文件**，元数据中必须有一列标记每个样本是 source 还是 sink。

```r
# ============================================================
# 文件：analysis/11_sourcetracker.R
# 功能：准备 SourceTracker 输入 + 可视化结果
# ============================================================

library(phyloseq)
library(dplyr)
library(readr)
library(tidyr)
library(ggplot2)

# 加载数据
ps <- readRDS("results/phyloseq_object.rds")
meta <- data.frame(sample_data(ps))

cat("样本分布：\n")
print(table(meta$body.site))
```

```
📊 输出：
样本分布：
      gut left palm right palm     tongue
       11         5          5          5
```

**我们的实验设计：** 肠道和舌头作为 source（已知的微生物组来源），左手掌和右手掌作为 sink（我们想知道它们的微生物从哪里来）。

```r
# ============================================================
# 构建 SourceTracker 的元数据
# 关键列：SourceSink（标记 source/sink）
#          Env（环境标签，source 的名称）
# ============================================================

st_meta <- meta %>%
    mutate(
        SampleID = rownames(meta),
        SourceSink = ifelse(
            body.site %in% c("gut", "tongue"),
            "source", "sink"
        ),
        Env = case_when(
            body.site == "gut" ~ "Gut",
            body.site == "tongue" ~ "Tongue",
            body.site %in% c("left palm", "right palm") ~ "Palm"
        )
    ) %>%
    select(SampleID, SourceSink, Env)

cat("Source/Sink 分布：\n")
print(table(st_meta$SourceSink, st_meta$Env))

# 保存元数据
write_tsv(st_meta, "results/sourcetracker_metadata.tsv")
```

```
📊 输出：
Source/Sink 分布：
          Gut Palm Tongue
  sink      0   10      0
  source   11    0      5
```

11 个肠道样本和 5 个舌头样本作为 source，10 个手掌样本（左右各 5）作为 sink。

## Step 2：运行 SourceTracker2

```bash
# ============================================================
# 运行 SourceTracker2
# --source_rarefaction_depth: source 样本稀释到的测序深度
# --sink_rarefaction_depth: sink 样本稀释到的测序深度
# --burnin: Gibbs 采样的 burn-in 轮数
# --restart: 随机重启次数（增加结果的稳定性）
# ============================================================

sourcetracker2 gibbs \
    -i results/picrust2_input/feature-table.biom \
    -m results/sourcetracker_metadata.tsv \
    -o results/sourcetracker_out \
    --source_rarefaction_depth 1000 \
    --sink_rarefaction_depth 1000 \
    --burnin 100 \
    --restart 10 \
    --jobs 4

echo "输出文件："
ls results/sourcetracker_out/
```

```
📊 输出：
输出文件：
mixing_proportions.txt
mixing_proportions_stds.txt
```

`mixing_proportions.txt` 是核心结果——每个 sink 样本中来自各 source 的估计比例。`mixing_proportions_stds.txt` 是对应的标准差。

⚠️ **踩坑预警：稀释深度的选择**

> SourceTracker 内部会对样本做稀释（rarefaction）。稀释深度需要在"保留足够的序列信息"和"确保所有样本都不被丢弃"之间平衡。
>
> 先用 `biom summarize-table` 查看每个样本的测序深度，然后选一个让 90% 以上样本都保留的深度。如果你设置的深度太高，低测序量的样本会被自动排除——SourceTracker 不会报错，只是结果表里少了几个样本。
>
> 我们选 1000 是因为 Moving Pictures 数据集的最小测序深度约 1100。正式项目中建议做多个稀释深度的敏感性分析。

```r

    left_join(
        st_meta %>% filter(SourceSink == "sink"),
        by = c("#SampleID" = "SampleID")
```

**结果非常清晰：手掌微生物组的绝大部分（~90%）是"Unknown"来源——既不来自肠道也不来自口腔。** 来自肠道的比例约 2-4%，来自口腔的比例约 4-7%。

这完全合理——手掌有自己独特的微生物组，以 *Staphylococcus*、*Propionibacterium* 等皮肤常驻菌为主，这些物种在肠道和口腔中都很少见。"Unknown"并不是"不知道"的意思，而是"来自我们没有定义为 source 的其他环境"——在这里就是皮肤微生物组本身。

## Step 4：左手 vs 右手的来源差异

一个有趣的小问题：左手和右手的来源比例有差异吗？

```r
# ============================================================
# 左手 vs 右手的来源比例比较
# ============================================================

source_comparison <- mixing_annotated %>%
    select(body.site, Gut, Tongue, Unknown) %>%
    group_by(body.site) %>%
    summarise(
        across(c(Gut, Tongue, Unknown),
               list(mean = mean, sd = sd)),
        .groups = "drop"
    )

cat("=== 左手 vs 右手 来源比例 ===\n")
print(source_comparison)

# Wilcoxon 检验
cat("\nGut 来源差异 (Wilcoxon p):",
    wilcox.test(
        mixing_annotated$Gut[mixing_annotated$body.site == "left palm"],
        mixing_annotated$Gut[mixing_annotated$body.site == "right palm"]
    )$p.value, "\n")

cat("Tongue 来源差异 (Wilcoxon p):",
    wilcox.test(
        mixing_annotated$Tongue[mixing_annotated$body.site == "left palm"],
        mixing_annotated$Tongue[mixing_annotated$body.site == "right palm"]
    )$p.value, "\n")
```

```
📊 输出：
=== 左手 vs 右手 来源比例 ===
  body.site  Gut_mean   Gut_sd Tongue_mean Tongue_sd Unknown_mean Unknown_sd
1 left palm   0.0294  0.00982     0.0520   0.01034      0.9186    0.02014
2 right palm  0.0316  0.00789     0.0525   0.00876      0.9159    0.01623

Gut 来源差异 (Wilcoxon p): 0.754
Tongue 来源差异 (Wilcoxon p): 0.917
```

**左手和右手的来源比例没有显著差异。** 这符合预期——同一个人的两只手暴露于相似的微生物环境，来源比例应该很接近。

💡 **经验之谈：SourceTracker 的最佳应用场景**

> Moving Pictures 数据集的溯源结果比较"无聊"——大部分是 Unknown。但 SourceTracker 在以下场景中非常强大：
>
> **环境微生物溯源：** 水源中的微生物有多少来自人类粪便污染？多少来自动物排泄物？多少是天然水体微生物？这在环境监测和公共卫生中至关重要。
>
> **医院感染溯源：** 患者术后感染的病原菌是来自手术器械、病房环境还是患者自身的菌群？
>
> **根际微生物溯源（和我们的研究更相关）：** 根际微生物有多少来自播种前的土壤？多少是从种子带来的？多少是灌溉水引入的？这对理解根际微生物的组装过程有重要意义。

## 方法论讨论：溯源分析的假设检验

SourceTracker 的结果好不好，很大程度上取决于你的 source 定义是否合理。这里有几个需要警惕的问题。

⚠️ **踩坑预警：Source 定义不全导致的误导**

> 如果你只定义了肠道作为 source，没有定义口腔，那手掌微生物中原本来自口腔的那 5% 会被错误地归入"Unknown"——或者更糟，被错误地归入肠道（如果口腔和肠道有少量共享物种）。
>
> **黄金法则：尽可能多地定义 source。** 如果你研究河流水质，source 应该包括：上游水、人类粪便、动物粪便、污水处理厂出水、农业径流等。遗漏一个 source 可能导致其他 source 的比例被高估。

⚠️ **踩坑预警：Source 样本量的不平衡**

> 我们有 11 个肠道样本但只有 5 个舌头样本作为 source。如果 source 之间样本量差异很大，SourceTracker 对高样本量 source 的估计会更稳定。
>
> 建议：尽量让各个 source 的样本量接近。如果实在不行，至少每个 source 要有 5 个以上样本。

## 本篇小结

这一篇我们用 SourceTracker2 对手掌微生物组进行了来源溯源。

**核心发现：** 手掌微生物组的约 90% 无法被肠道或口腔来源解释——这些是皮肤自身的常驻微生物。口腔来源约占 5%，肠道来源约占 3%，这个微小但可检测的贡献可能反映了日常的手-口和手-消化道接触。左手和右手的来源比例无显著差异。

**方法层面的关键收获：**

1. **Source 的定义决定了溯源的质量。** 遗漏重要的 source 会导致错误归因。
2. **"Unknown"不是失败——它是信息。** 高比例的 Unknown 说明 sink 环境有自己独特的微生物组。
3. **稀释深度需要仔细选择。** 太高会丢样本，太低会丢信号。做敏感性分析是好习惯。
